(1) The current code is only showing 3x3

(2) Editing cells is not resposive.

When editing a cell it should pop up the line being edited up in the formula bar, and update that bar with every key press.

(3) It should have a menu bar accessed with the ALT key.

(4) It should have a status line at the bottom, suggesting context sensitve hot keys  the user might want to press.
